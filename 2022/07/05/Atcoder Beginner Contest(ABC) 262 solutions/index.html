Atcoder Beginner Contest(ABC) 262 solutions

Author: Ivan Li Date: July 5, 2022  0:00:00

check the link: https://atcoder.jp/contests/arc141/tasks

Alert: I think the first few problems really do not deserve thorough analysis, so I am going to put my comment in the code

A - World Cup

B. Triangle(Easier)

C - Min Max Pair

D - I Hate Non-integer Number

this is a pretty interesting question.

First of all, we need to locate the algorithm: dynamic programming, because it is an obvious linear relationship

so how do we define the dp equation?

2 dimensions are clear: choose j items from the first i items

the third dimension should help form the equation.

as we want the sum to be divisible, we will think of modulo

now a hypothesis: dp(i,j,k) : choose j items from first i items, and sum mod j = k, the total number of approaches

we happily discover that this perfectly suits our need, as we therefore has the dp equation:

dp(i,j,k) = dp(i-1, j, k) + dp(i, j-1, (k- a[i] % num + num) % num)

dp(i-1, j, k) is when I don’t choose a[i]

dp(i, j-1, (k- a[i] % num + num) % num is when I choose a[i], so the first j-1 items need to fulfill certain requirements

That’s all for this update! The following four harder problems will be explained in detail in the next update!
